A Machine Learning project that classifies sonar signals as either rocks or mines using Logistic Regression. The project includes complete data preprocessing, model training, evaluation, and accuracy testing on the Sonar dataset.

Key Features:

Dataset: Sonar signals dataset (Rocks vs Mines)

Algorithm: Logistic Regression

Data preprocessing with NumPy and Pandas

Model evaluation using accuracy score and confusion matrix

Well-documented and beginner-friendly implementation

Tech Stack:
Python, Scikit-learn, NumPy, Pandas
